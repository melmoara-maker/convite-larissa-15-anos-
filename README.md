# Convite completo — 15 anos da Larissa

Projeto pronto para GitHub Pages.

## Alterações desta versão

- Ao tocar em **Abrir convite**, a abertura leva diretamente para a página da mensagem.
- O botão **Ver localização** abre uma janela dentro do convite.
- A janela de localização possui:
  - botão X para fechar;
  - botão “Fechar e voltar ao convite”;
  - link para abrir o endereço no Google Maps.
- Ao fechar, o visitante volta ao mesmo ponto do convite.
- Áudio enviado pelo WhatsApp mantido.
- Lista de presentes, confirmação de presença e demais animações mantidas.
